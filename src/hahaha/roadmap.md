# 路线图与任务清单

基于 `repo/Hahaha/doc/zh-cn/Task.md` 整理的开发进度：

## ✅ 已完成 (v0.0.1 阶段)
- [x] **张量核心**：维度管理、内存所有权、嵌套初始化、广播适配。
- [x] **线性代数基础**：矩阵乘法 (`matmul`)、转置 (`transpose`)、求和 (`sum`)。
- [x] **自动微分引擎**：动态计算图节点、拓扑排序、反向传播。
- [x] **优化器基础**：SGD 随机梯度下降。
- [x] **可视化**：基于 ImGui 的训练曲线实时绘制。
- [x] **基础设施**：日志系统、Docker 开发环境、Meson 构建流。

## 🚧 进行中 (v0.1.0 目标)
- [x] **广播机制完善**：支持更复杂的跨维度张量运算。
- [ ] **数据集实现**：创建数据集抽象接口，完成数据集加载的任务。
- [ ] **现代 ML 模型**：线性回归、KNN 模板实现。
- [x] **日志增强**：集成 `std::stacktrace` 提供奔溃现场追溯。
